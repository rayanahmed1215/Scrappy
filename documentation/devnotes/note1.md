Notes on Scrappy design
Core Goal (MVP)

Scrappy’s MVP goal is to function as an AI prompt routing system.

The system receives a user prompt and routes it to an AI model using a Smart Router, which is currently based on embedding similarity logic.

High-Level System Direction

Scrappy is evolving into a system that combines:

Prompt routing
Context awareness
Memory-aware decision making
Modular execution pathways

At MVP stage, only routing is required, but the design is intentionally extensible.
